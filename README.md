# PrettyLittleThingsTX 

*This is a E-commerce website I am working on for a client who owns a small business. It is built in HTML,CSS/SCSS, and Javascript.*


[Live Preview](https://josephrgonzalez.github.io/pretty-little-things-tx/)
![snap1](/preview/doc-img/01_preview.png)



### Features

- Clean CSS structure 
- Clean markup **(Markup correctly formatted with comments for setup)**.
- Fully responsive **(For desktop and mobile users)**.
- Cross Browser compatibility **(Work on a variety of browsers)**.
- Fresh Design **(Modern and aesthetic approaches used by many web apps)**.
- User convenient **(Easy to use for any professional and non-professional)**.
- W3C Validated **(All files are passed from W3C validation service)**.
- Well Documented **(Files are commented)**.
